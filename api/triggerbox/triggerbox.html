事件触发框

说明:

触发框（triggerbox）用于自定义功能，图标可以自定义。2019-01-03

如：

默认触发框
图片触发框
卡触发框
复制触发框 
<script type="text/javascript">
	function msg(value,name){
		var title = "点击了name="+name+"触发组件的按钮,value="+value;
		$.messager.popover({type:"info",msg:title});
		if(name=="copy"){
			var v = $("#card").triggerbox('getValue');
			$("#copy").triggerbox('setValue',v);
		}
	}
</script>
<table cellpadding="5">
	<tr>
		<td class='r-label'>默认触发框</td>
		<td>
			<input name="default" class="hisui-triggerbox textbox" data-options="handler:msg"/>
		</td>
	</tr>
	<tr>
		<td class='r-label'>图片触发框</td>
		<td>
			<input name="img" class="hisui-triggerbox textbox" data-options="icon:'icon-w-img',handler:msg"/>
		</td>
	</tr>
	<tr>
		<td class='r-label'>卡触发框</td>
		<td>
			<input name="card" class="hisui-triggerbox textbox" data-options="icon:'icon-w-card',handler:msg"/>
		</td>
	</tr>
	<tr>
		<td class='r-label'>复制触发框</td>
		<td>
			<input name="copy" class="hisui-triggerbox textbox" data-options="icon:'icon-w-copy',handler:msg"/>
		</td>
	</tr>
</table>
属性 说明 默认值
icon 可配置按钮显示的图标 'icon-w-trigger-box' 见【基础】-【图标(icon)】界面
方法 说明 入参
getValue 获得组件的值 无
setValue 给组件赋值 val
事件 说明 入参
handler 点击按钮时触发事件 value,name
